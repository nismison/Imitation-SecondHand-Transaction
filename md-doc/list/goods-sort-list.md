### 商品列表（一行一行的列表）

****

##### 参数说明

| 参数 | 类型 | 默认 | 说明 |
| --- | --- | --- |
| list_data | Array | 无 | 列表数据 |
| show | Boolean | true | 是否显示 |
| @listTap | 事件 | (data,index) | 被点击 |

****

##### list_data 参数说明

| 参数 | 类型 |  说明 |
| --- | --- | --- |
| quv | Boolean | 是否已验机 |
| autarky | Boolean | 是否自营 |
| type | Array | 商品规格标签 |
| price | String | 价格 |
| cost_price | String | 原价 |
| title | String | 商品标题 |
| img | String | 商品图片 |
| discount | Array | 价格后的小标签 |
| style | Array | 底部小标签 |
